## Full description (до 4000 символов)

```
Stocksi Ultimate — новостной агрегатор для активных инвесторов. Приложение
показывает поток биржевых новостей в реальном времени с моментальной
доставкой через WebSocket-канал.

Что внутри:
• Лента новостей по российским и зарубежным эмитентам — котирующимся на
  Мосбирже, СПБ Бирже, NYSE, NASDAQ.
• Источники — десятки информационных агентств, корпоративных пресс-релизов,
  макроэкономических индикаторов. Можно включать/выключать любые группы.
• Фильтры — чёрные и белые списки тикеров, хэштегов, ключевых фраз.
  Можно отфильтровать только дивиденды или только санкции.
• Подсветка текста — собственный DSL: задаёшь правила вида «дивиденды@green»
  и слово автоматически выделяется в каждой новости. Поддерживается полный
  набор CSS-свойств для текста (цвет, фон, font-weight, размер, подчёркивание).
• Звуковые алерты — отдельные звуки для разных тегов: например, отчётность
  играет одним звуком, санкции — другим.
• Push-уведомления — приходят даже когда экран заблокирован (опционально,
  с фоновым режимом).
• Тёмная и светлая темы — переключаются одной кнопкой в нижней панели.
• Свайп-жесты — поделиться, скопировать, открыть оригинал, вернуться назад.

Приложение не оказывает финансовых услуг, не выполняет сделок, не даёт
торговых рекомендаций. Это исключительно средство просмотра новостной ленты.

Доступ к ленте — по токену, который выдаётся через Telegram-бот
@StocksiUltimate_bot. Подписка приобретается там же.

Сайт: stocksi-ultimate.ru
Поддержка: @StocksiUltimate_bot
```

## Иконка 512×512

Бери из `flutter-app/assets/icon/app_icon.png` (1024×1024) и ресайзни до
512×512. Можно через PowerShell:

```powershell
Add-Type -AssemblyName System.Drawing
$src = "flutter-app/assets/icon/app_icon.png"
$dst = "play-icon-512.png"
$bmp = New-Object System.Drawing.Bitmap 512, 512
$g = [System.Drawing.Graphics]::FromImage($bmp)
$g.SmoothingMode = "AntiAlias"
$g.InterpolationMode = "HighQualityBicubic"
$g.DrawImage([System.Drawing.Image]::FromFile($src), 0, 0, 512, 512)
$bmp.Save($dst, "Png")
```
