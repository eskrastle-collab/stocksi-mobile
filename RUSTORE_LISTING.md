## RuStore Console шаги

### 1. Кабинет → «Приложения» → «+ Создать приложение»

**Тип**: Бесплатное (или Платное если будешь продавать прямо в магазине; у нас оплата идёт через Telegram-бот, поэтому **Бесплатное**).

**Категория**: **Новости и журналы**.

**Имя приложения**: `Stocksi Ultimate`

**Уникальное имя (package name)**: `ru.stocksi.ultimate` (должно совпадать с APK).

### 2. Описание

**Краткое описание** (до 80 знаков):
```
Лента биржевых новостей в реальном времени с подсветкой и фильтрами
```

**Полное описание** (до 4000 знаков):
```
Stocksi Ultimate — новостной агрегатор для активных российских инвесторов.
Приложение показывает поток биржевых новостей в реальном времени с
моментальной доставкой через защищённый WebSocket-канал.

Что внутри:

• Лента новостей по российским и зарубежным эмитентам — котирующимся на
  Мосбирже, СПБ Бирже, NYSE, NASDAQ.
• Десятки источников — информационные агентства, корпоративные
  пресс-релизы, макроэкономические индикаторы. Любые группы можно
  включить или выключить в настройках.
• Гибкие фильтры — чёрные и белые списки тикеров, хэштегов, ключевых
  фраз. Хотите видеть только дивиденды? Только санкции? Настраивается.
• Подсветка текста — собственный DSL: задаёшь правила вида «дивиденды@green»
  и слово автоматически выделяется в каждой новости. Поддержка цвета,
  фона, размера и веса шрифта.
• Звуковые алерты — отдельные звуки для разных категорий новостей.
  Например, отчётность играет одним звуком, санкции — другим.
• Push-уведомления — приходят даже когда экран заблокирован.
• Тёмная и светлая темы.
• Свайп-жесты — поделиться, скопировать, открыть оригинал, вернуться назад.

Приложение не оказывает финансовых услуг, не выполняет торговых операций
и не даёт инвестиционных рекомендаций. Это исключительно средство
просмотра новостной ленты для информированных решений.

Доступ к ленте — по токену, который выдаётся через Telegram-бот
@StocksiUltimate_bot. Подписка приобретается там же.

Сайт: stocksi-ultimate.ru
Поддержка: @StocksiUltimate_bot
```

**Ключевые слова**: `биржа, новости, акции, тикер, торги, мосбиржа, инвестор, рынок`

### 5. Раздел «Сбор данных» / «Соответствие 152-ФЗ»

RuStore спросит обрабатываем ли персональные данные пользователей.

**Ответ**: **Не обрабатываем**.

Обоснование (если попросят):
- Все настройки и токен хранятся локально на устройстве пользователя
- Никаких аналитик / трекеров / рекламных SDK
- WS-подключение к серверу новостей — только для получения контента,
  не передаётся персональная информация

### 6. APK

Загружаешь `Ultimate-rustore.apk`. RuStore автоматически распарсит
applicationId, версию, разрешения.

**Замечание про разрешения**: у нас в манифесте есть `POST_NOTIFICATIONS`,
`FOREGROUND_SERVICE`, `FOREGROUND_SERVICE_DATA_SYNC`, `WAKE_LOCK`,
`VIBRATE`. Для каждого RuStore попросит обоснование — формулировки:

| Разрешение | Обоснование |
|---|---|
| INTERNET | Подключение к серверу новостей |
| POST_NOTIFICATIONS | Уведомления о новых новостях |
| FOREGROUND_SERVICE / FOREGROUND_SERVICE_DATA_SYNC / WAKE_LOCK | Поддержание WebSocket-соединения когда экран заблокирован — без этого Android выгружает приложение и пользователь пропускает свежие новости |
| VIBRATE | Вибрация при пуше (опционально) |

### 7. Иконка

Загружаешь `Ultimate-play-icon-512.png` (512×512 PNG).

### 8. Скриншоты

Минимум 2, желательно 3-5. Размер 1080×1920 или больше.

Идеи кадров (сделать прямо с телефона Power+VolumeDown в Ultimate):
1. **Лента новостей в тёмной теме** с разноцветными чипами тикеров и подсветкой ключевых слов.
2. **Настройки → Фильтры** — чёрный/белый списки, секция подсветки с DSL.
3. **Детальный просмотр** новости с full-text.
4. **Настройки → Уведомления** — toggle алертов и звуков.
5. **Светлая тема** — для контраста, чтобы магазин видел два режима.

### 9. Видео (опционально)

RuStore разрешает добавить видеообзор (YouTube link или mp4). Не
обязательно для модерации, но повышает конверсию установок.

Можно добавить позже.

### 10. Нажать «Отправить на модерацию»

Срок проверки **1-3 рабочих дня** обычно. Если найдут что-то — придёт
e-mail с указанием что доработать. Перезаливаешь APK или меняешь текст
и отправляешь снова.

## Подводные камни

1. **Финансовые приложения**. RuStore *возможно* спросит лицензию ЦБ если позиционировать приложение как финансовый сервис. Текст выше явно говорит «не оказывает финансовых услуг» — этого достаточно.

2. **Работа без подключения**. Может попросить пояснение почему приложение требует интернет всегда. Ответ: «лента живая, без подключения нет смысла».

3. **Доступ через токен**. Скорее всего попросят тестовый токен. Создай отдельный токен для модератора через своего бота — пометь как «reviewer-token». После прохождения модерации можешь его отозвать.

## Дальше — обновления через RuStore

Поднимаешь `versionCode` в `pubspec.yaml`, пересобираешь APK, заходишь
в RuStore Console → твоё приложение → «Версии» → «Загрузить новую
версию» → ждёшь 1-2 дня модерации. После — раскатывается у пользователей
автоматически (RuStore сам обновит).
