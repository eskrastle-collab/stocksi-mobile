Политика конфиденциальности

Последнее обновление: 4 мая 2026 г.

Stocksi Ultimate (далее — «Приложение») — новостной агрегатор для русскоязычных инвесторов. Приложение разрабатывается командой Stocksi.

Настоящая политика описывает, какие данные использует Приложение, для чего и кому они передаются.

1. Какие данные мы собираем

Приложение не собирает персональные данные пользователей. В частности, мы НЕ запрашиваем и НЕ передаём третьим лицам:

  • ФИО, дату рождения, паспортные данные;
  • адреса (физические, почтовые, e-mail);
  • список контактов, переписку, фотографии;
  • идентификаторы устройства (IDFA / Android Advertising ID);
  • геолокацию;
  • данные об использовании приложения (аналитика).

2. Какие данные хранятся локально

Для работы Приложение хранит только на устройстве пользователя:

  • Токен доступа (UUID), который пользователь самостоятельно вводит при первом запуске. Токен выдаётся через Telegram-бот @StocksiUltimate_bot и используется только для авторизации на сервере новостей.
  • Пользовательские настройки — выбранные источники, фильтры, правила подсветки текста, выбранный звук уведомлений, тема оформления.
  • Кэш новостей, полученных от сервера — для отображения после перезапуска приложения.

Все эти данные хранятся в защищённом каталоге Приложения, не передаются на наши серверы и не покидают устройство. Токен авторизации хранится в системном хранилище ключей (Android Keystore / iOS Keychain) и не доступен другим приложениям.

3. Сетевые соединения

При работе Приложение устанавливает соединения с:

  • wss://websocket.priority.news — WebSocket-канал, по которому сервер шлёт новости. Передаётся: токен авторизации, технические heartbeat-пакеты. Не передаётся никакая информация об устройстве, помимо стандартных HTTP-заголовков (User-Agent).
  • Внешние ссылки в текстах новостей и в иконках тикеров — открываются по запросу пользователя в браузере или загружаются как обычные изображения. Никаких track-параметров мы не добавляем.

Все соединения используют шифрование TLS 1.2+.

4. Уведомления и сообщения об ошибках

Приложение использует системные уведомления Android / iOS и звуки уведомлений. Это происходит локально, без участия каких-либо внешних push-сервисов (Firebase Cloud Messaging и т. п.).

Для повышения качества Приложения мы используем сервис мониторинга крашей. При непредвиденной ошибке программы отправляется обезличенный технический отчёт: тип ошибки, версия приложения, версия операционной системы, модель устройства. Личные данные, содержимое новостей, токен авторизации в этом отчёте не передаются.

5. Передача данных третьим лицам

Stocksi Ultimate не передаёт пользовательские данные третьим лицам.

6. Удаление данных

Чтобы полностью удалить данные Приложения с устройства:

  1. Откройте «Настройки» → «Приложения» → «Stocksi Ultimate» → «Хранилище» → «Очистить данные».
  2. Или удалите Приложение целиком.

Поскольку никаких данных Приложение не передаёт на наши серверы, отдельного запроса на удаление не требуется.

7. Дети

Приложение не предназначено для детей младше 13 лет.

8. Изменения политики

Мы можем обновлять эту политику. Текущая версия всегда доступна по адресу stocksi-ultimate.ru/privacy.

9. Контакты
